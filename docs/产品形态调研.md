# 以太产品形态调研

## 结论

以太当前版本不再采用长期悬浮桌面宠物作为主入口，改为“主界面 + 系统托盘 + 全局热键 + 短答案卡”的按需问答器。普通 Electron 窗口无法可靠覆盖独占全屏游戏，长期悬浮会带来打不开、抢焦点、遮挡和反作弊兼容风险。

## 主流形态

- Overwolf：官方公开约 1.13 亿月活，采用后台运行、游戏内 Overlay、热键召唤和按需显示。它适合商业化游戏内 Overlay，但需要 Overwolf 客户端、开发者审批和发布流程。
- Xbox Game Bar：Windows 原生游戏小组件入口，强调 `Win+G` 热键、固定卡片、点击穿透和有限尺寸。
- Discord、Steam、NVIDIA：均采用热键打开 Overlay，不承诺普通桌面窗口长期压在所有游戏之上。
- Gaming Copilot：以热键、语音、按需屏幕理解和短反馈为主，不把常驻宠物作为核心交互。

## 对以太的取舍

- 当前作品版优先稳定展示“遇到问题 → 按 Alt+Q → 以太看懂画面 → 给出短建议 → 展示完整 trace”。
- 桌面宠物降级为品牌头像，只用于托盘、主界面和答案卡。
- Overwolf 保留为后续适配层，不作为当前交付依赖。

# 后台（AgentOps）操作手册

后端排障与复盘入口：用于查看会话、技能、来源、Trace 与错误日志，支持历史管理。

## 主要区域

- 会话列表：按账号/UID 和最近更新时间排序  
- 详情区：选择某一会话后，自动加载该会话对应的最新一轮 run
- 标签页建议：
  - 总览：最终回复、最近消息、关键指标
  - 视觉观察：截图识别、视觉特征、置信度
  - 知识来源：本地卡片命中、web 检索来源、过滤来源
  - Skill：技能链路与触发原因
  - Trace：完整时间线与调用阶段
  - 错误：失败记录、规则、日志

## 标准排障闭环

### 1. 打开历史会话
1. 在会话列表选中目标会话  
2. 右侧详情必须同步为该会话内容（而非始终显示最新会话）  
3. 验证“当前会话 ID”与页面上下文一致

### 2. 查看来源与决策依据
1. 打开 `知识来源`，核对是否展示：
   - 具体检索关键词
   - 来源 URL
   - 是否有来源过滤
2. 打开 `Skill`，确认命中/跳过原因
3. 打开 `Trace` 查看每一步耗时

### 3. 继续追问（后台）
1. 选中会话后输入问题并发送  
2. 结果应追加到该会话，不创建新会话  
3. 如有失败，记录到错误 tab，结合 Trace 回放

### 4. 历史管理
- 单条删除：删除会话与相关运行快照/历史记录  
- 按账号清空：清空该 UID 下所有会话（可选同时清空 memory）  
- 全部清空：清空全局历史（操作前弹窗确认）

## 录屏演示建议（1-2 分钟）

1. 进入会话列表，点开一条含历史回复的记录  
2. 展示右侧联动后的 `trace + knowledge + skill`  
3. 再在该会话追问一次“再给我一个保守方案”  
4. 演示失败/重试逻辑与清理按钮

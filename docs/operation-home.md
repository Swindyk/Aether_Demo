# 首页操作手册（玩家视角）

本页用于演示、答辩或日常使用：从“打开助手”到“继续追问”一条龙闭环。

## 适用场景

- 想快速知道当前卡点怎么走
- 想在不离开游戏的情况下获取配队/机制建议
- 想复用当前上下文继续问下一步

## 5 分钟演示流程（推荐）

### 0:00 - 0:20
1. 打开以太，确认位于“首页”。
2. 简单说明：这是一个本地桌面 AI 伴侣，主入口是 `Alt+Q`，并且支持后续“继续追问”。

### 0:20 - 1:40
1. 点击场景卡（如“配队要不要换/剧情讲到哪/探索卡在哪”）。  
2. 仅做**问题偏好选择**，不触发识别。  
3. 若希望立即解读，点击主按钮（或按 `Alt+Q`）。

### 1:40 - 3:00
1. 观察截图来源与解读结果。  
2. 强调“先基于可见信息给出两步建议”，再落到可执行动作。

### 3:00 - 4:30
1. 在“继续追问”中输入一句追问（例如“下一步建议优先做什么”）。  
2. 发送后，确认页面先显示用户问题，再显示“以太思考中...”  
3. 拿到回答后再进行下一问。

### 4:30 - 5:00
1. 简要切到后台入口说明（见后台手册）。  
2. 说明会话与 trace 便于校验“为什么这么建议”。

## 实际操作步骤

### 第一步：打开应用
- 打开以太后先看首页按钮和状态，确认截图来源不是空态。
- 先关闭会话提示遮挡，确保游戏窗口可见。

### 第二步：开始一次解读
- 选择场景：`探索卡在哪` / `这队能不能打` / `装备要不要换` / `剧情讲到哪`（只改问题，不执行解读）
- 点击 `看当前画面` 或按 `Alt+Q` 启动识别
- 默认返回：结论 + 下一步建议（两步内）

### 第三步：继续追问
- 在“继续追问”输入框中提问  
- 点击发送后，保持会话上下文（同一会话）  
- 如果未及时返回，等待“以太思考中...”状态，不要重复点击

## 注意点（易错点）

- 点击场景卡不应触发识别。  
- 只有按下 `Alt+Q` / 主按钮才会抓取当前画面。  
- 如果截图源错误，多半是窗口焦点问题，先切换到游戏窗口再试。

# 以太作品集包装

## 简历标题

以太 AI 游戏伴侣｜面向玩家卡点场景的桌面端屏幕理解 Agent

## 一句话定位

面向开放世界和 RPG 玩家，在装备、配队、剧情、探索卡点时按 `Alt+Q` 读取当前画面，给出结论优先的短建议，并用 AgentOps 展示真实后台链路。

## 投递方向

- AI 产品：多模态 Agent、RAG、memory、工具链路、产品边界。
- 游戏产品：玩家卡点、攻略查询成本、低打扰交互、账号公开数据增强。
- 游戏运营：知识库更新、攻略来源治理、版本内容同步。
- 系统或工具策划：热键、托盘、短答案卡、AgentOps 证据链。

## 作品集 4 页结构

1. 玩家痛点：卡点时切屏查攻略会打断游戏体验。
2. 产品方案：`Alt+Q` 看当前画面，短答案卡给一句结论和两步行动。
3. Agent 架构：截图理解、memory、SQLite 知识、Tavily、skill、rules、真实请求编号。
4. 边界与迭代：不读内存、不抓包、不自动代玩；后续可接 Overwolf、语音和更完整知识库。

## 不夸大口径

- 不说官方数据接入，只说公开 UID 和本地知识库。
- 不说独占全屏全覆盖，只建议无边框窗口模式。
- 不说自动代玩，只给手动建议。
- 不报没有评测依据的准确率或用户指标。

# 首页操作手册（玩家视角）

本页用于演示、答辩或日常使用，覆盖从打开助手、选择问题偏好、触发画面解读到继续追问的一条完整闭环。

## 适用场景

- 想快速知道当前卡点下一步怎么走。
- 想在不离开游戏的情况下获取配队、装备、机制或路线建议。
- 想复用当前截图和会话上下文继续追问。
- 想演示“低打扰屏幕 Agent”的玩家侧体验。

## 5 分钟演示流程

### 0:00 - 0:20：打开首页

1. 启动以太，确认主界面停留在“首页”。
2. 简短说明：以太是一个本地桌面 AI 游戏助手，核心入口是 `Alt+Q`，也可以点击首页“让我看看”。
3. 说明短答案卡会在分析后自动出现，完整回答可回到首页查看。

### 0:20 - 1:20：选择问题偏好

1. 点击一个场景卡，例如“装备要不要换”“这队能不能打”“剧情人物是谁”或“探索卡在哪”。
2. 说明场景卡只改变本轮问题倾向，不会立即截图或发起模型调用。
3. 选择一个回答偏好，例如“新手玩家”“进阶玩家”或“剧情玩家”，说明同一张截图会根据偏好改变解释深度和关注点。

### 1:20 - 2:40：触发画面解读

1. 切到目标游戏、窗口或演示画面。
2. 按 `Alt+Q`，或点击首页“让我看看”。
3. 观察状态从“正在捕获画面”进入“正在分析”，再出现短答案卡。
4. 强调回答目标是“先看懂当前可见信息，再给两步以内的下一步建议”。

### 2:40 - 4:10：查看完整回答并继续追问

1. 回到首页，查看“最近一次回答”。
2. 在“继续追问”输入框里输入一句追问，例如“保守一点下一步先做什么”。
3. 发送后确认页面先追加用户问题，再显示“以太思考中”状态，最后把回答追加到同一会话。

### 4:10 - 5:00：切到后台验证链路

1. 点击右上角后台入口进入 AgentOps。
2. 展示本轮会话的视觉观察、知识命中、Skill、Trace 和错误信息。
3. 说明后台用于解释“为什么以太会这样建议”，玩家正常使用时不需要打开。

## 实际操作步骤

### 第一步：确认画面来源

- 默认设置是读取鼠标所在显示器。
- 如果默认截图源不对，在首页选择目标显示器、目标窗口或内置演示画面。
- 演示录屏时建议使用无边框窗口模式，避免独占全屏或受保护画面被截成黑屏。

### 第二步：选择问题倾向

- 场景卡：`装备要不要换` / `这队能不能打` / `剧情人物是谁` / `探索卡在哪`。
- 回答偏好：`随心玩家` / `进阶玩家` / `剧情玩家` / `新手玩家` / `收集玩家`。
- 这些选择只影响下一次回答的侧重点，不会单独触发截图。

### 第三步：开始一次解读

- 点击“让我看看”或按 `Alt+Q`。
- 等待短答案卡出现。
- 首页“最近一次回答”会显示更完整的结论、当前队伍/画面判断、建议和依据。

### 第四步：继续追问

- 在“继续追问”输入框中输入问题。
- 追问会复用当前会话、上一轮观察和已命中的知识，不会重新截图。
- 如果没有创建过会话，需要先按 `Alt+Q` 或点击“让我看看”完成一次解读。

## 注意点

- 点击场景卡不应触发识别；只有 `Alt+Q` 或“让我看看”才会截图。
- `Alt+Shift+Q` 用于打开最近一次完整回答。
- 如果快捷键冲突，可以直接点击首页按钮。
- 如果截图源错误，先把鼠标移到目标显示器，再按 `Alt+Q`；仍不正确时手动选择显示器或窗口。
- 如果截图为黑屏，优先切换无边框窗口模式，并检查系统屏幕录制/窗口捕获权限。

## 口播建议

- “以太不是自动操作游戏，而是把当前画面看懂后给下一步建议。”
- “场景卡只是告诉它我要问什么，真正截图发生在 Alt+Q 或点击按钮之后。”
- “短答案卡解决当下操作，完整回答和后台用于复盘依据。”
